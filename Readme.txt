Web Based Calculator 

-enter numbers on thekeypad and add them to a list (you can clear it anytime)

-display the current number (clear it anymtime)

-sum the list numbers in the list among each other

-calculate the mean value of the numbers in the list 

-calculate the mean value of the squared numbers in the list 

-ignore a chosen number

-calculate the variance of the numbers in the list

Made with jQuery, Bootstrap, HTML by Claudio Perizzolo (Klod19)

Enjoy!
